DOM: Document Object Model is the programming interface through which we can manipulate HTML. Basically no one has authority over HTML. But with the help of DOM we can change these various objects, properties, methods, events, styles etc.

Select elements by DOm: There are 5 ways to select elements by DOM:
a) getElementsByTagName()
b) getElementsById()
c) getElementsByClassName()
d) querySelector()
e) querySelectorAll()

Event bubble: When an event handler is applied to an element, it is applied to its parent, then to its parent, and so on until it reaches the end.

Callback Function: It is a function that has some specific actions inside it and when called from outside with a value to use it, it completes its action and returns the value.
